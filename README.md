# SQL Data Analysis – Task 3

## Description
This project demonstrates SQL data analysis using:
- JOINs
- GROUP BY
- HAVING
- Subqueries

## Database Schema
- students(id, name)
- courses(id, name)
- enrollments(id, student_id, course_id, grade)

## Analysis Performed
1. Top student per course
2. Pass rate per course (grade ≥ 40)
3. Overall topper across all courses
4. Students enrolled in multiple courses
